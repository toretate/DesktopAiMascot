# マスコット表情スプライト自動位置合わせ仕様書

## 1. 概要

### 1.1 背景

マスコット画像から、顔部分を拡大して最大16種類の表情スプライトを作る処理は作成済みです

* マスコット詳細 - 表情アセット - AI表情作成 を実行

このスプライト作成後、16種類の表情スプライトを区分けして、各表情スロットに設定する処理も作成済みです

* マスコット詳細 - 表情アセット - AIスプライトインポート を実行

作成されたスプライトをそのまま使うには下記の問題があります

* 表情ラベルや、区分けに使った枠
* 眉、目、口の表情が必要だが、不要な部分まで含まれている
* 表情画像の拡大率がマスコット画像と違うため、拡大 もしくは 縮小しないと合わない

この 自動位置合わせは下記の手順を実行することで自動的に表情スプライトを最適な位置に移動します


1. 表情スプライトの顔矩形の把握
  1. 表情スプライトの顔（眉、目、口、その他必要な装飾を含んだ矩形）位置の把握
  2. 表情スプライトに対して上記矩形外をマスク領域として設定し、表情は顔だけの表示とする

2. 表情スプライトの拡大 or 縮小
  1. マスコット側の顔位置の把握
  2. 顔の輪郭位置や、目の大きさや位置などから、表情スプライトが合うように scale を算出

3. 表情スプライトの位置設定
  1. 表示されている表情スプライトが合うようにスプライトの位置を設定

## プログラム構成

言語: Typescript
※サーバ側実装の場合、PythonコードをCallするのは有り
※顔位置検出ライブラリの導入もOK

## 特記事項

マスコット画像は、一般には正面で立っていたり、バストアップが使われるが下記のパターンも将来的考慮が必要になる

* 体の向きが横：マスコットーの体が横を向いて、顔がすこしこちらを向いている
  * 例：箒を持って顔だけこちらを少し向いている
* 違う体制：マスコットが座っている、布団に寝ている
  * 椅子に座る、机に頬杖をついてこちらを見ている
  * 体の向きが横 とのコンビネーション


## テストファイル

__tests__/expresssion-alignment/assets/ テスト用のディレクトリ
  * ${outfit}/ マスコット毎のディレクトリ
    * outfit_xx.png : マスコット画像
    * expr_${表情}.png : 表情スプライト画像
    * expr_${表情}_trimmed.png : 表情スプライトの顔部分を手作業で設定し、マスキングした画像
    * expr_${表情}_OK.png : trimmed の表情スプライトをマスコット画像に手作業で重ねて保存した画像
