# 表情エディタ＆位置調整ダイアログのウィザード化

## 概要

既存の `ExpressionEditorModal.vue` を、表情スプライト作成からパーツ抽出、瞬き・口パクアニメーション確認、最終アトラス（SpriteSheet）エクスポートまでをシームレスに行える「4ステップのウィザード形式」に改造する。

## 目的

- モーダル内に `currentStep` 状態を導入し、以下の4ステップのウィザードを構築する。
    1. **ベース顔確認**: のっぺらぼう画像の仕上がりプレビューとレタッチ・再生成確認。
    2. **表情位置調整**: 既存のスライダーUI（`offsetX`, `offsetY`, `scale`, `rotation`）を流用し、感情スロット別のアライメント調整を行う。
    3. **アニメーション確認**: 分離された目・口のパーツを使用した瞬き・口パクの動作プレビュー（ループアニメーション表示）。
    4. **アトラスエクスポート**: 結合アトラス画像（`atlas.png`）と `atlas.json` のパッキングプレビューおよび保存。
- ユーザーが必要に応じて手動介入し、設定を調整できるようにする。

## 完了条件

- `ExpressionEditorModal.vue` が 4ステップのウィザード形式で動作する。
- スライダー調整および感情切り替えが正常に動作する。
- アニメーションプレビューパネルで瞬き・口パク動作を確認できる。
- 保存処理の実行により、正しいアトラス画像と JSON が書き出される。
- 変更後のコードビルドおよびテストが通る。
