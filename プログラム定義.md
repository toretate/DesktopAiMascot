# プログラム定義
デスクトップマスコットを表示して、ユーザーと対話するプログラムを作成します。このプログラムは、以下の機能を持ちます。

* デスクトップ上にマスコットキャラクターを表示する。
	* 画像サイズは 1024x768 ピクセル
	* 画像フォーマットは PNG または WebP。透過背景をサポートする。
* マスコットキャラクターは、ユーザーのマウスクリックやキーボード入力に反応する。
* マスコットをドラッグして移動できる。
	* 前回の位置を記憶し、次回起動時に同じ位置に表示する。
* システムトレイに常駐し、右クリックメニューから表示/非表示や終了ができる。


# コントロール配置

* アプリ全体として MascotForm を用意。ドラッグすると移動可能にする。
    * 背景を半透明に設定する
* 右半分に MascotControl を配置。マスコットの描画とアニメーションを担当。
    * 背景は透過に設定する
* 左半分に InteractionPanel を配置。ユーザーとの対話を担当。
    * 背景を透過に設定する
	* ユーザーとの対話処理は、LINEの用に吹き出し形式で表示する
	* 最下にユーザーのテキスト入力欄を用意する。SHIFT+ENTERキーで送信する
	* キャラクターとの対話メッセージは、LM Studio を用いて生成する
	* LM Studio との通信は、HTTP API 経由で行う。Endpointは http://127.0.0.1:1234 です

# 対話処理
* IAiChatService インターフェースを定義し、対話処理を抽象化する
	* SendMessageAsync(string message): ユーザーからのメッセージを送信し、キャラクターからの応答を非同期に取得する
	* 取得したメッセージは InteractionPanel に表示する
* IAiChatService の実装として LmStudioChatService クラスを作成し、LM Studio との通信を実装する
	* HttpClient を用いて、LM Studio の HTTP API にリクエストを送信する
	* レスポンスからキャラクターの応答メッセージを抽出、返却する
