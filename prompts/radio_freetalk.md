# フリートーク シナリオ生成プロンプト

## 概要
このプロンプトは、2人のパーソナリティによる自然なフリートーク（雑談）のシナリオを生成します。
決まったテーマはなく、日常的な話題で楽しく会話する内容を作成します。

---

## プロンプト本文

```
あなたは、ラジオ番組の台本作家です。
2人のパーソナリティによる自然なフリートークのシナリオを作成してください。

【登場人物】
- メインパーソナリティ（MC）: 明るく話好きで、話題を次々と展開していく
- サポート役（アシスタント）: リアクション上手で、話を膨らませる役割

【フリートークのコンセプト】
- 特定のテーマは決めず、自然な雑談を展開
- 日常的で親しみやすい話題を選ぶ
- リスナーも共感できる内容を心がける
- ユーモアと温かみのある雰囲気を作る

【シナリオの条件】
1. 会話は12〜18ターン程度（1人あたり6〜9回発言）
2. 話題は自然に転換していく（2〜3つの小話題を含む）
3. 笑いや驚き、共感などの感情表現を豊かに
4. 「そういえば」「ところで」などの自然な話題転換を使用
5. 最後は明るく締めくくる

【話題の例】
- 最近あった面白いこと、嬉しかったこと
- 日常の小さな発見や気づき
- 趣味や好きなことについて
- 季節の風物詩
- 身近な出来事や体験
- ちょっとした疑問や不思議

【会話の流れ（目安）】
1. 軽い導入（最近の出来事など）
2. 第1話題の展開と盛り上がり
3. 自然な話題転換
4. 第2話題の展開
5. （必要に応じて第3話題）
6. 楽しい締めくくり

【出力形式】
以下のJSON形式で出力してください：

{
  "scenario": "freetalk",
  "title": "フリートーク",
  "topics": ["話題1", "話題2", "話題3"],
  "conversations": [
    {
      "speaker": "MC",
      "text": "発言内容",
      "emotion": "happy/excited/surprised/thoughtful/amused",
      "topic_index": 0
    },
    {
      "speaker": "Assistant",
      "text": "発言内容",
      "emotion": "happy/impressed/amused/gentle/excited",
      "topic_index": 0
    }
  ]
}

【注意事項】
- 各発言は1〜3文程度の自然な会話調にしてください
- 相槌や感嘆詞（「へぇー！」「そうなんだ」等）を効果的に使用
- 話題転換は自然に行い、唐突にならないように
- 2人のキャラクターの違いを意識した発言にする
- ネガティブな話題や論争的な話題は避ける
- リスナーを意識した発言（「みなさんもそうですよね？」等）を適度に含める
```

## カスタマイズオプション

### 時間帯による話題の調整
```
【追加条件】
- 現在の時間帯: {morning/afternoon/evening}
- 時間帯に合った話題を選んでください
```

### 季節に合わせた話題
```
【追加条件】
- 現在の季節: {spring/summer/autumn/winter}
- 季節感のある話題を1つ以上含めてください
```

---

## 実装上の注意

1. **話題のバランス**
   - 重すぎず軽すぎないバランスを保つ
   - 1つの話題に偏りすぎないよう注意

2. **自然な会話のリズム**
   - 一方的にならないよう、2人のバランスを取る
   - 適度な間や相槌を含める

3. **topic_index の活用**
   - 話題の切り替わりポイントを明確化
   - UI上で現在の話題を表示可能
